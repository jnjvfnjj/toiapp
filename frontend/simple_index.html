Выполните следующие шаги для сборки:

  1. Откройте терминал в папке проекта
  2. Перейдите в папку frontend: cd frontend
  3. Установите зависимости: npm install
  4. Соберите приложение: npm run build
  5. Обновите эту страницу

Или запустите dev сервер для разработки:

cd frontend

npm install

npm run dev

Приложение будет доступно на http://localhost:3000
